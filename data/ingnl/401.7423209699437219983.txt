# Machtigen

## Hoe meld je een gemachtigde af?

### Hoe kan ik een machtiging afmelden?

Een gemachtigde afmelden kan online via de ING Mobiel Bankieren App of in Mijn ING. Via het overzicht zie je alle afgegeven machtigingen. Je kan er één of meerdere intrekken. De gemachtigde heeft dan binnen één tot vijf werkdagen geen toegang meer.
Lukt dit niet? Dan kan het ook via een ING-kantoor of -servicepunt geregeld worden. **Let op:** op ING-kantoren kun je alleen nog langskomen met een afspraak. Check in de locatiezoeker welk ING-kantoor bij jou in de buurt zit en plan een afspraak in . Voor ING-Servicepunten is geen afspraak nodig.

### Met deze stappen meld je een machtiging af via de ING Mobiel Bankieren App

1. Open de App. Klik rechtsonder op 'Service';
2. Klik op 'Je rekening';
3. Klik op 'Machtiging instellen';
4. Klik op ‘Gemachtigde afmelden’.

### Met deze stappen meld je een machtiging af via Mijn ING

1. Direct een gemachtigde afmelden via Mijn ING
2. Login op Mijn ING;
3. Open het overzicht Machtigingen;
4. Klik op ‘Gemachtigde afmelden’.

### Een machtiging afmelden via een formulier

Heb je geen toegang tot Mijn ING en ben je niet in staat om langs te komen op een ING-kantoor of -servicepunt? Dan kun je de machtiging ook afmelden via het formulier 'Afmelden gemachtigde'. Dit formulier kun je ook gebruiken als je jezelf als gemachtigde wil afmelden. Denk er dan wel aan dat je de rekeninghouder hierover informeert.
