# Machtigen

## Veelgestelde vragen

### Hoe kan ik een gemachtigde Betaalpas beëindigen?

Een rekeninghouder of de gemachtigde kan een gemachtigde Betaalpas beëindigen. Dit kan je doen via Mijn ING of de Mobiel Bankieren App. Lukt dit niet, dan kan je dit ook op een ING-kantoor of -servicepunt laten doen.

# Machtigen

## Veelgestelde vragen

### Kan een gemachtigde ook meer bevoegdheden krijgen?

Je kan als gemachtigde beperkt handelingen verrichten. Een rekeninghouder zou er voor kunnen kiezen om dit via een notariële volmacht uit te breiden. Klik hier voor meer informatie over wettelijke vertegenwoordiging.

### Kan een gemachtigde het (correspondentie) adres van een rekeninghouder wijzigen?

Nee, een gemachtigde heeft niet de bevoegdheid om een (correspondentie) adres van de rekeninghouder te wijzigen. Een rekeninghouder dient deze handeling zelf te doen.

### Kan een gemachtigde (kopie) Afschriften of Jaaroverzichten opvragen?

Nee, een gemachtigde kan geen (kopie) Afschriften of Jaaroverzicht opvragen. Je dient dit als rekeninghouder zelf te doen omdat hier kosten aan zijn verbonden. De gemachtigde kan via Mijn ING wel zelf afschriften downloaden. Dit is kosteloos.

### Wat is de verantwoordelijkheid/bevoegdheid van de rekeninghouder?

De rekeninghouder is verantwoordelijk voor de handelingen van de gemachtigde. De rekeninghouder kan de machtiging intrekken.
