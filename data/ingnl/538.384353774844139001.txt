# Contant geld opnemen met je telefoon - Cashcode - ING

## Hoe werkt het?

### Hoe werkt de Cashcode op de geldautomaat van Geldmaat?

* Kies op het startscherm van de geldautomaat van Geldmaat voor 'Zonder Betaalpas'.
* Maak de keuze voor 'ING code'.
* Voer je Cashcode in op de geldautomaat.
* Volg daarna de stappen in het scherm op de geldautomaat.
Een uitgebreide beschrijving en schermen van de geldautomaat, kun je vinden via de website van Geldmaat.

### Kan iedereen gebruik maken van deze optie?

Het aanmaken van een Cashcode via de ING Mobiel Bankieren App is op dit moment alleen mogelijk voor particuliere klanten van minimaal 18 jaar oud.

### Voor wie is opnemen zonder pas bedoeld?

Opnemen met de Cashcode is bedoeld voor particuliere klanten (18+) met een behoefte aan contant geld zonder (werkende) pas bij zich. Dit kan zijn omdat er een nieuwe pas in aanvraag is, maar ook in geval de pas niet meer gebruikt wordt en jij alles met de mobiele telefoon betaalt.

### Hoelang is de Cashcode geldig?

De Cashcode is maximaal 1 uur geldig. Zodra de code gebruikt is, is deze niet meer geldig.
Wanneer je code niet meer geldig is, dan kun je wel opnieuw een nieuwe Cashcode aanvragen

### Op welke momenten kan een Cashcode aangemaakt worden?

De Cashcode kan op ieder moment aangemaakt worden, echter omdat de geldautomaten ’s nachts niet geopend zijn, is de code alleen binnen de openingstijden van de geldautomaten te gebruiken.

### Hoe vaak kan de Cashcode gebruikt worden?

Een Cashcode kan slechts 1 keer gebruikt worden. Het is wel mogelijk een nieuwe Cashcode aan te maken. Opnemen met de code kan alleen wanneer de limiet van € 100 per dag voor opnemen met de Cashcode nog niet benut is.

### De transactie is afgebroken, kan ik de Cashcode weer gebruiken?

Een Cashcode kan slechts 1 keer gebruikt worden. Het is wel mogelijk een nieuwe code aan te maken. Opnemen met de code kan alleen wanneer de limiet van € 100 per dag nog niet benut is.
